# AutoVen Web

AutoVen 智能运营助手的前端界面。

## 快速开始

### 安装依赖

```bash
npm install
# 或
pnpm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
npm start
```

## 功能

- 💬 **对话界面** - 流式输出，实时显示 AI 思考过程
- ⏰ **定时任务** - 管理计划任务
- 📁 **工作区** - 查看已生成的笔记和内容
- ⚙️ **设置** - 管理用户人设和偏好

## 技术栈

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Zustand
- Radix UI
- Lucide Icons
